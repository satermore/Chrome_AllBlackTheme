##ALLBALCK THEME FOR CHROMIUM

A sleek dark theme for Chrome and Brave, featuring a fully black interface with inactive windows in gray. Designed to enhance window movement with the mouse while keeping the browser visually clean and easy on the eyes.

## Features

- **Full black interface:** everything in pure black for a modern, minimalist look.
- **Inactive windows in gray:** clearly distinguish between active and inactive tabs/windows.
- **Enhanced window movement:** optimized for smooth dragging and resizing with the mouse.
- **Eye-friendly:** reduces glare and distractions while browsing.
